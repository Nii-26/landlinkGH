# LandLinkGH

A full-screen, mobile-first web mapping application.

## Description

This project provides an interactive web-based mapping application designed with a mobile-first approach and full-screen display capabilities. It enables users to explore geographic data and locations in an intuitive, responsive interface.

## Features

- Full-screen mapping experience
- Mobile-first responsive design
- Interactive geographic visualization

## Technologies

- HTML5

## Getting Started

To use this application, simply open it in a web browser to view and interact with the map.
